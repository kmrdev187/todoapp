## Project setup

### Clone repo

```shell
> git clone https://github.com/kmrdev187/todoapp.git
```

### Install project dependencies

```shell
> yarn install
```

### Compiles and hot-reloads for development

```shell
> yarn electron:serve
```

### Compiles and minifies for production

```shell
> yarn electron:build
```
